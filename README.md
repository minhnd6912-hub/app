# Healthcare App - Quản lý Lịch Hẹn

## Giới thiệu
Ung dụng quản lý lịch hẹn khám chữa bệnh với các chức năng:
- Quản lý lịch hẹn
- Quản lý khách hàng
- Thanh toán
- Quản lý thu chi
- Chương trình giám giá
- Quản lý lịch làm việc

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js + Express
- **Database**: MySQL

## Cấu trúc dự án
```
healthcare-app/
├── frontend/          # React app
├── backend/           # Node.js API server
└── docs/              # Tài liệu
```

## Cài đặt

### Prerequisites
- Node.js v14+
- npm hoặc yarn
- MySQL 5.7+

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
